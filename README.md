# Forevo

Полноценный мессенджер с real-time общением. Работает локально и с двух устройств в одной Wi-Fi сети.

## Возможности

- Регистрация и вход
- Личные и групповые чаты
- Real-time сообщения (WebSocket)
- Статус «в сети» / «был(а) недавно»
- Индикатор «печатает...»
- Прочитано (галочки)
- Ответ на сообщение
- Редактирование и удаление
- Реакции (эмодзи)
- Отправка фото и файлов
- Голосовые сообщения (удерживайте 🎤)
- Поиск пользователей
- Профиль с именем и описанием
- Адаптивный интерфейс (телефон + ПК)

## Быстрый старт

### 1. Установка

```bash
npm run install:all
```

### 2. Запуск

```bash
npm run dev
```

Откроется:
- **Сервер:** `http://localhost:3001`
- **Клиент:** `http://localhost:5173`

### 3. Два устройства

1. Запустите сервер на ПК (команда выше).
2. В консоли сервера будет **Network URL**, например: `http://192.168.1.5:5173`
3. На втором устройстве (телефон/планшет) откройте этот адрес в браузере.
4. Оба устройства должны быть в **одной Wi-Fi сети**.
5. Зарегистрируйте два разных аккаунта и начните чат через «✏️» → поиск пользователя.

> Если не открывается — разрешите порты 3001 и 5173 в брандмауэре Windows.

## Продакшен (один порт)

```bash
cd client && npm run build
cd .. && npm start
```

Откройте `http://<ваш-IP>:3001` на обоих устройствах.

## Деплой на Render

1. Загрузите код на GitHub
2. Создайте **New Web Service** на [render.com](https://render.com)
3. Подключите репозиторий
4. Render автоматически использует `render.yaml` — создаст веб-сервис + бесплатную PostgreSQL базу
5. Переменная `DATABASE_URL` подключится автоматически
6. Добавьте `JWT_SECRET` в переменные окружения (нажмите "Generate Value")

## Стек

- **Backend:** Node.js, Express, Socket.io, PostgreSQL
- **Frontend:** React, Vite
